2. استفاده از دانش کسب شده برای فعالیت‌های غیرقانونی یا مخرب کاملاً ممنوع است.

3. آموزشگاه اختلال هیچ گونه مسئولیتی در قبال استفاده نادرست از مطالب آموزشی ندارد.

4. شركت كنندگان متعهد مي‌شوند كه از اطلاعات ارائه شده فقط براي مقاصد قانوني و اخلاقي استفاده كنند.

5. تمامی محتوای آموزشی متعلق به آموزشگاه اختلال است و هرگونه کپی‌برداری بدون مجوز ممنوع است.

6. آموزشگاه در قبال هرگونه آسیب مستقیم یا غیرمستقیم ناشی از استفاده از آموزش‌ها مسئولیتی ندارد.
